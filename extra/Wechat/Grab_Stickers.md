# mac微信获取缓存内容

## 需要
* 系统: Mac OS
* 如需获取动态表情，需要旧版本Mac微信如(2.2.7)

## 步骤
* Finder -> 前往文件夹(Command+Shift+G)
```text
~/Library/Containers/com.tencent.xinWeChat/Data/Library/Application Support/com.tencent.xinWeChat/
```

* 在用户文件夹(有点像随机值的文件夹)中查找自己所需的内容
  * `Stickers` 是表情文件夹
  * ...
