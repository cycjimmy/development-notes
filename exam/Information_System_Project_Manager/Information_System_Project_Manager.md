# 信息系统项目管理师

## 考试介绍
### 文件依据
依据人事部、信息产业部文件（国人部发【2003】39号）
* 证书全国范围有效
* 取得高级资格，可聘任高级工程师职务
* 以考代评

### 考试科目及形式
| 考试科目 | 形式 | 考试时长 | 合格标准 |
| :------: | :------: | :------: | :------:|
| 综合知识 | 75道单选题（总分75分） | 2.5小时(09:00-11:30) | 45分 |
| 案例分析 | 3道案例分析（每题25分） | 1.5小时(13:30-15:00) | 45分 |
| 论文 | 二选一，2800字左右 | 2小时(15:20-17:20) | 45分 |
