# Localtunnel

## 使用
* 默认安装了node的环境: [https://nodejs.org](https://nodejs.org)
* 安装localtunnel
  ```shell
  $ npm install -g localtunnel
  ```

* 映射本地端口到域名
  ```shell
  $ lt --subdomain [个性前缀] --port [要映射的端口]
  ```
