# Charles抓包工具

## 下载与安装
* 官网下载链接：https://www.charlesproxy.com/download/
* 注册
  * 打开charles -> `help` -> `register`
  * 使用 [Charles 激活码计算器](https://www.zzzmode.com/mytools/charles/) 生成License
  * 将生成的 `Registered Name` 和 `License Key` 填入
  * 重启软件
