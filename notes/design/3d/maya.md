# Maya

## 操作
### 快捷键
* `F`: 快读定位
* `W`: 移动工具
* `E`: 旋转工具
* `R`: 缩放工具
* `command + D`: 复制

### 导出fbx
* File -> Export All

### 重新链接丢失文件
* Windows > General Editors > File Path Editor
* Auto Resolve
